# 🎮 Programação Avançada em Java – Desenvolvimento de Jogo 2D
Este repositório contém os exercícios e desafios desenvolvidos durante as aulas de **Programação Avançada em Java**.

O foco das aulas é aprender **Programação Orientada a Objetos (POO)** aplicada à criação de **interfaces gráficas**, utilizando **painéis, molduras e janelas**, que servirão como base para o desenvolvimento de um **jogo 2D no projeto final da disciplina**.

O objetivo deste repositório é registrar minha **evolução durante as aulas**, documentando cada desafio e aprendizado ao longo do semestre.

# 🧠 Conteúdos aprendidos

Durante as aulas, estamos aprendendo conceitos importantes de **Java orientado a objetos aplicados a interfaces gráficas**.

## Estrutura de aplicações gráficas

Criação de aplicações utilizando:

- Janelas
- Molduras
- Painéis

Esses elementos são fundamentais para a construção da interface de um jogo.

---

## Organização em classes

O projeto é dividido em diferentes responsabilidades:

### Classe Principal
Responsável por iniciar a execução do programa.

### Classe Moldura (Frame)
Responsável pela criação da janela principal da aplicação.

### Classe Painel (Panel)
Área onde os elementos gráficos e componentes serão desenhados.

Essa separação ajuda a manter o código **mais organizado e reutilizável**.

# 🎮 Projeto Final da Disciplina

O objetivo final da disciplina é desenvolver um **jogo 2D em Java** utilizando os conceitos aprendidos ao longo das aulas.

Para isso, estamos construindo gradualmente a base do jogo, aprendendo:

- Estrutura de janelas
- Renderização em painéis
- Organização do código orientado a objetos
- Base para lógica de jogos
